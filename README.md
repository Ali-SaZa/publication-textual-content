# publication-textual-content
A simple application for publishing text content.

## Requirements

node version: ^14.18.0

npm version: ^6.14.15

## Installation Packages

Use the package manager [npm](https://www.npmjs.com/) to install packages.

```bash
npm install
```

## Run Development

```bash
npm run serve
```

## Build

```bash
npm run build
```
